# 足関節・足部（整形外科 診察テンプレート Oのみ）

- 視診：Barefoot, Swelling, Redness, Deformity, Valgus/Varus/Flat/Cavus/HV/Claw, Arch, Gait
- Tenderness：MM-/+ LM-/+ ATFL-/+ CFL-/+ Deltoid-/+ Syndesmosis-/+ Talus-/+ Achilles-/+ Calcaneus-/+ Navicular-/+ Cuboid-/+ 5thMT-/+ Lisfranc-/+ PF-/+
- Ankle ROM：DF 20/20, PF 45/45, Inv 30/30, Ev 20/20
- Great toe ROM：MTP Flex 35/35, MTP Ext 60/60, IP Flex 60/60, IP Ext 0/0, Abd 20/20, Add 10/10
- Lesser toe ROM：MTP Flex 35/35, MTP Ext 40/40, PIP Flex 35/35, PIP Ext 0/0, DIP Flex 50/50, DIP Ext 0/0
- 神経循環：Sensation, Motor, Circulation
- 追加身体所見：one finger test（圧痛局在） Heel raise可/不可  single heel rise（hindfoot varus化）-/+
- 特殊テスト（疑う疾患に応じて選択）
  - 外側靱帯：AD（anterior drawer）-/- TT（talar tilt）-/-
  - 高位足関節捻挫：Squeeze-/- ER stress-/- Cotton-/-（必要時）
  - アキレス腱：Thompson（Achilles）-/-（必要時）
  - 足底腱膜：Windlass（PF）-/-（必要時）
  - Lisfranc：Piano key（midfoot）-/-（必要時）
  - 足部アライメント/拘縮：Silfverskiold-/-
- ＜略語＞
  - MM: Medial malleolus, LM: Lateral malleolus, ATFL: Anterior talofibular ligament, CFL: Calcaneofibular ligament
  - DF: Dorsiflexion, PF: Plantarflexion, Inv: Inversion, Ev: Eversion
  - TT: Talar tilt, ER stress: External rotation stress test
