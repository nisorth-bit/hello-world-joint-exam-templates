## 内装・動線設計の要点

### 必要スペース

| 室 | 必要性 | 備考 |
|---|---|---|
| 待合室 | 必須 | 車椅子・松葉杖対応 |
| 受付・事務 | 必須 | |
| 診察室 | 必須 | 1〜2室 |
| X線室 | 必須 | 鉛壁・防護扉 |
| 処置室 | 必須 | ギプス・小外科対応 |
| リハビリ室 | 検討 | PT配置の場合 |
| 手術室（外来手術） | 検討 | 将来拡張 |
| スタッフルーム | 必須 | |
| トイレ | 必須 | バリアフリー対応 |

### 動線の考え方

- 患者動線：入口 → 受付 → 待合 → 診察 → 処置・X線 → 会計 → 出口（逆流なし）
- スタッフ動線：受付 ↔ 診察室 ↔ 処置室（短距離）
- 感染対策：発熱患者用の別動線を検討
